# Design Principles

- Every screen must answer one primary question
- Avoid cognitive overload
- Prefer whitespace over borders
- Animation should reinforce meaning
- Keyboard-first interactions
- Fast interactions over flashy visuals
- Minimal color palette
- Maximum 2 focal points per screen
- No generic AI gradients
- No unnecessary cards

# Motion Philosophy

Animation should:
- reduce cognitive friction
- improve clarity
- reinforce hierarchy
- create tactile feedback
- reward progress

Avoid:
- flashy effects
- excessive parallax
- distracting motion
- gimmicky 3D

Motion style:
- calm
- intelligent
- smooth
- subtle
- premium

Reference products:
- Linear
- Arc
- Raycast
- Apple
- Vercel

Use:
- Framer Motion for UI transitions
- Respect reduced-motion accessibility settings

UI principles:
- minimal
- calm
- high information clarity
- premium spacing
- subtle depth
- motion-driven feedback
- keyboard-first
