# Product Vision
A productivity system that feels intelligent, calm, fast, and adaptive.

# Core Philosophy
Reduce friction.
Reduce cognitive overload.
Make progress addictive.

# Stack
- React 18 + Vite + TypeScript
- TailwindCSS v4 + Framer Motion
- Zustand (Local UI State)
- TanStack Query (Server State)
- Supabase (Backend/Auth)
- React Router DOM (Routing)

# UI Direction
Linear + Arc + Notion inspired.
Premium.
Minimal.
Keyboard-first.
Dark mode.
Touch-friendly mobile layouts.

# Current Features
- Magic Link & Google SSO Authentication
- Global Quick Capture (⌘K)
- Task Management with Optimistic Updates
- Minimalist Focus Timer with ambient breathing animations
- Responsive Sidebar & Mobile Bottom Navigation Bar
