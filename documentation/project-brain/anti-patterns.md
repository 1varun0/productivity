# Anti Patterns

- No cluttered dashboards
- No unnecessary graphs
- No excessive notifications
- No gradient spam
- No tiny unreadable text
- No fake gamification
- No 20-feature sidebars
- No productivity guilt UX
- No loading spinners where optimistic UI is possible
- No jumpy layout shifts during loading
- No generic AI SaaS visual aesthetics
- No borders where whitespace suffices
