# Peak Hub Keyboard Shortcuts

Peak Hub is designed for high-performance navigation and structural alignment. To maximize efficiency, the following global and contextual keyboard shortcuts have been implemented across the application.

## Global Shortcuts

| Shortcut | Action | Context |
| :--- | :--- | :--- |
| `CTRL + SPACE` | Quick Task Capture | Opens the Capture Modal from anywhere. |
| `CTRL + SHIFT + SPACE` <br/> `CTRL + SHIFT + P` | Global Mental Capture | Dumps a thought into the Mental Inbox. |
| `CTRL + K` | Command Palette | Access navigation, tasks, and quick actions. |
| `ALT + N` | New Note | Automatically opens Nexus and creates a new note. |
| `CTRL + /` | Shortcuts Reference | Opens the visual shortcuts reference sheet. |
| `F` | Enter Focus Mode | Enters the Focus Chamber (disabled when typing). |
| `ESC` | Close Modal | Gracefully closes any active modal or panel. |

## Feature-Specific Shortcuts

### Nexus (Markdown Editor)
| Shortcut | Action |
| :--- | :--- |
| `CTRL + B` | Bold text `**text**` |
| `CTRL + I` | Italicize text `*text*` |
| `CTRL + SHIFT + C` | Format as inline code `` `code` `` |

### Command Palette
| Shortcut | Action |
| :--- | :--- |
| `Arrow Up` / `Arrow Down` | Navigate list |
| `Enter` | Execute selected action |
| `ESC` | Close palette |

### Focus Chamber (Orb Ritual)
| Shortcut | Action |
| :--- | :--- |
| `SPACE` | Pause / Resume focus session |
| `ESC` | End session early / Close custom input |
| `Number Keys (0-9)` | Input custom temporal extension (minutes) |
| `ENTER` | Submit temporal extension |
| `BACKSPACE` | Remove last digit in custom input |

### Timetable
| Shortcut | Action |
| :--- | :--- |
| `CTRL + Z` | Undo last grid action (if not typing in an input) |

### Task Details Panel
| Shortcut | Action |
| :--- | :--- |
| `ESC` | Close the panel, close link composer, or close image lightbox |
| `CTRL + ENTER` | Save task changes and blur input |

### Thought Parking (During Focus)
| Shortcut | Action |
| :--- | :--- |
| `ESC` | Close thought parking modal |
| `ENTER` | Save parked thought to session context |

> **Note**: For macOS users, `CTRL` shortcuts map directly to the `CMD` key.
